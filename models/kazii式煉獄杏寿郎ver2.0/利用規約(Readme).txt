補足
★「鼻穴線消し」モーフ→エッジ太くすると鼻の穴のエッジが目立つ時があるのでその対策用です。
★技術不足の為、物理演算が脆いです。羽織がひどく破綻・貫通した場合そのまま公開しないでください。
出来る限り直すか、その部分を映さないか、分割して撮るか、羽織なしの隊服モデルを使うか、モデル自体を使わないか、対応お願いします。
腕の貫通は肩腕を少し下げたり前に出したりすると耐える事も多いので試してみてください。
★「口角下げ」→口角が下がってます。それ以外は通常モデルと一緒です。
基本顔の口角がかなり上がってるので真顔の場合モーフの口角下げだけでは厳しいと考え分けて作りました。
ただ口角下げを意識しすぎたのか若干怒り顔です。
